# macOS Interface Recreation

A web application that recreates the macOS Sequoia/Sonoma desktop interface using Vue 3 with Composition API.

## Features

- **Menu Bar**: Top bar with Apple logo, menu items, and system icons
- **Dock**: Bottom dock with app icons and hover effects
- **App Drawer**: Click the app drawer icon (ðŸ“±) in the dock to open a drawer with more applications

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown in the terminal (usually `http://localhost:5173`)

## Build

To build for production:
```bash
npm run build
```

## Tech Stack

- Vue 3 (Composition API with `<script setup>`)
- Vite
- Regular CSS (no frameworks)
